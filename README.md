# Urban Analysis Tool

QGIS-плагин для пространственного анализа городской инфраструктуры: оценка доступности социальных объектов для жилой застройки и проверка санитарно-защитных зон (СЗЗ) промышленных предприятий.

**Автор:** Шапошникова А.С.  
**Версия:** 0.1  
**QGIS:** 3.0+

## Возможности

### Коэффициент доступности инфраструктуры

- Выбор типа объектов с нормативным радиусом (школы, детсады, поликлиники, аптеки, площадки, транспорт или произвольный радиус).
- Расчёт расстояния до ближайших объектов в метрах (корректно для географических и проекционных СК).
- Классификация: «Хорошо» / «Удовлетворительно» / «Недостаточно».
- Результирующий слой с атрибутами и цветовой символикой.
- Сводный отчёт по доле домов с достаточной доступностью.

### Проверка СЗЗ

- Радиусы по классу опасности (`hazard_class`, 1–5) согласно СанПiН 2.2.1/2.1.1.1200-03.
- Поиск домов в зоне с пространственным индексом и точной проверкой расстояния в метрах.
- Слои: полигоны СЗЗ (полупрозрачные) и точки нарушений.
- Отчёт и лог по заводам с классами и дистанциями.

## Требования к данным

- Слой жилой застройки и слой объектов — **одна система координат**.
- Для СЗЗ у промышленных объектов поле **`hazard_class`** (целые значения 1–5).
- Поддерживаются точечные и полигональные слои (для расчёта используются центроиды).

## Установка

### Из исходников (разработка)

1. Клонируйте репозиторий:

```bash
git clone https://github.com/YOUR_USERNAME/urban_analysis_tool.git
```

2. Скопируйте папку плагина в каталог QGIS:

```
%APPDATA%\QGIS\QGIS3\profiles\default\python\plugins\urban_analysis_tool
```

(на Linux/macOS: `~/.local/share/QGIS/QGIS3/profiles/default/python/plugins/`)

3. В QGIS: **Модули → Управление модулями → Установленные** — включите **Urban Analysis Tool**.

### Сборка ресурсов (при изменении `resources.qrc` / иконок)

```bash
make compile_resources
```

или через `pb_tool` / `pyrcc5`, если настроен Makefile.

## Использование

1. Откройте QGIS и загрузите слои жилой застройки и объектов инфраструктуры / промпредприятий.
2. **Модули → Urban Analysis Tool → Анализ городской инфраструктуры**.
3. Выберите режим анализа, слои и параметры, нажмите **OK**.
